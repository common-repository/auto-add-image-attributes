== Description ==

Automatically add image's Title, Caption, Alt Text and Description from image's filename with this WordPress plugin.

Just activate this plugin and it will automatically add image's Title, Caption, Alt Text and Description from image's filename for newly uploaded images.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/auto-add-image-attributes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. All done!
